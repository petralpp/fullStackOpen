# fullStackOpen

Ratkaisut Helsingin yliopiston Full Stack Open -kurssin ydinkurssin (osat 0-6) ja lisäosien (osat 6 ja 7) tehtäviin.

## Osa 0 - Web-sovellusten toiminnan perusteet

Web-sovellusten perusteista, HTTP-protokolla ja selain-palvelin-kommunikaatio, DOM, SPA

## Osa 1 - Reactin perusteet

Reactin alkeet, JavaScript, komponentin tila ja tapahtumankäsittely, Reactin debuggaus

## Osa 2 - Palvelimen kanssa tapahtuva kommunikointi

Reactin kokoelmien renderöinti ja moduulit, lomakkeiden käsittely, tyylit, palvelimella olevan datan haku ja muokkaus

## Osa 3 - Palvelimen ohjelmointi NodeJS:n Express-kirjastolla

Node.js ja Express, REST API, pyyntöjen testaus, middlewaret, MongoDB, validointi ja ESLint, deployaaminen

## Osa 4 - Express-sovellusten testaaminen, käyttäjänhallinta

Backendin yksikkö- ja integraatiotestaus, Mongoose, token-perustainen kirjautuminen

## Osa 5 - React-sovelluksen testaaminen

Kirjautuminen frontendissä, local storage, ehdollinen renderöinti, Vitest, e2e-testaus ja Cypress/Playwright

## Osa 6 - Edistynyt tilan hallinta

Tilan hallinta Reduxilla, Redux Toolkit, React Query, useReducer ja context

## Osa 7 - React router, custom-hookit, tyylikirjastot ja webpack

React Router, custom-hookit, valmiit käyttöliittymätyylikirjastot, webpack, Reactin luokkakomponenteista ja tietoturvasta
