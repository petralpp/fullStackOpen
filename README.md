# fullStackOpen

Ratkaisut Helsingin yliopiston Full Stack Open -kurssin ydinkurssin (osat 0-5) ja lisäosien (osat 6 ja 7) tehtäviin.

## Osa 0 - Web-sovellusten toiminnan perusteet

- Web-sovellusten perusteista
- HTTP-protokolla ja selain-palvelin-kommunikaatio
- DOM
- SPA

## Osa 1 - Reactin perusteet

- Reactin alkeet
- JavaScript
- Komponentin tila ja tapahtumankäsittely, debuggaus

## Osa 2 - Palvelimen kanssa tapahtuva kommunikointi

- Reactin kokoelmien renderöinti ja moduulit
- Lomakkeiden käsittely
- Tyylien lisäys
- Palvelimella olevan datan haku ja muokkaus

## Osa 3 - Palvelimen ohjelmointi NodeJS:n Express-kirjastolla

- Node.js ja Express
- REST API ja pyyntöjen testaus
- Middlewaret
- MongoDB
- Validointi ja ESLint
- Sovelluksen deployaaminen

## Osa 4 - Express-sovellusten testaaminen, käyttäjänhallinta

- Backendin yksikkö- ja integraatiotestaus
- Mongoose
- Token-perustainen kirjautuminen

## Osa 5 - React-sovelluksen testaaminen

- Kirjautuminen frontendissä, local storage
- Ehdollinen renderöinti
- Vitest
- E2E-testaus, Cypress/Playwright

## Osa 6 - Edistynyt tilan hallinta

- Tilan hallinta Reduxilla
- Redux Toolkit
- React Query, useReducer ja context

## Osa 7 - React router, custom-hookit, tyylikirjastot ja webpack

- React Router
- Custom-hookkien teko
- Valmiit käyttöliittymätyylikirjastot
- Webpack
- Reactin luokkakomponenteista ja tietoturvasta
